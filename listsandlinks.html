Creating a New Webpage

  1. Open up a template of an html file and copy.
  2. Create two new files called (name).html and (name).css.
  3. Paste the template into the html file you just created as a guideline for your document.
  4. Paste this element under meta charset:link rel="stylesheet" href="(name).css" type="text/css" to link the new css with your new document.
  5. Edit your document, making sure that it remains validated everytime you edit it. Use different elements such as "h1", "h2", "p", "header", "footer", ect.
  6. If you've finished editing your html file, you can move on to the (name).css file to style the page.
  7. Make sure that you are closing your tags in the css file, and choosing the right selectors.
  8. When you are finished editing your css and html files, make sure to add, commit, and push to github.
  9. To view the file in the production server, open a new tab in terminal and enter the command "python -m SimpleHTTPServer"
  10. The display should say "Serving HTTP on 0.0.0.0 port 8000"
  11. In the search bar on a browser, type "localhost:8000/(html name)
  12. You are now viewing it from the development server.
  13. In terminal, enter the sftp command: "sftp (name1234)-hsteacher".
  14. Enter yes, and your password for sftp.
  15. CD to the public_html and use "put (filename)" to save your file into the folder.
